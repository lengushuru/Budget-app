# 📖 Smart Budget <a name="about-project"></a>

Smartbudget is a mobile web application where users can manage their budget: user have a list of transactions associated with a category, so that user can see how much money he/she spent and on what.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running follow these simple example steps.

### Prerequisites

In order to run this project you need:

- Code Editor (Vs Code) or any editor of your choice

- Terminal

- Install Ruby on your computer

- Install Rails on your computer

### Setup

Clone this repository by using the command line to your desired folder:  

```sh
git clone  https://github.com/lengushuru/Budget-app.git

cd budget-app
```

### Install

Install this project with:

```sh
cd budget-app
 bundle install (install all dependencies)
```

### Usage

To run the project, execute the following command:

```sh
rails server
```

### Test

```sh
  rspec
```
